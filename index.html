铁男对线思路

Posted on 2024-09-03

前中后期出装及相关思路

  1. 多兰盾+血瓶出门
  2. 五鞋子选一，看对线英雄
  3. 如果对上鳄鱼等物伤回血英雄，速出宝珠或小反甲
  4. 核心四件按顺序出：
    1. 裂隙制造者出完有续航
    2. 冰杖出完具有粘人能力
    3. 背包出完后，具备初步开团能力，利用背包主动触发冰杖后，配合ER拿下对方大哥，为打团创造优势，在大招结束前观察红点，提前释放技能（对面看不到不好躲）
    4. 大后期考虑把重伤合完，或者大帽
      ==逆风局考虑做完鬼书，大后期换大帽==

对线思路

  • 对面牛逼：起手E，利用反向E技巧把对面推开后补兵
  • 对面垃圾：起手Q，Q他妈的
  • 通用：
    1. 善用W抵消对方关键伤害技能
    2. 在上路6级后能单杀时，利用闪现或走位调整身位至对方英雄身后，使得防御塔——你——对方英雄形成一条线，而后放R，防止大招范围落在地方防御塔下
    3. 在对方有能力格挡大招时，善用E骗取对方格挡技能
    4. 牢记q技能伤害范围是高略大于技能指示器、底边在远处的等腰三角形，判定范围与技能前摇和指示器都略有差别，可用于远程消耗
    5. q的cd极短，亮了就点
